## Definition

- SPA: clientside
- MPA: server side

```
╭───────────╮ ────> ╭──────────────────╮ <──── ╭─────────────╮
│ JS - Code │ ────> │ HTML - Structure │ <──── │ CSS - Style │
╰───────────╯ ────> ╰──────────────────╯ <──── ╰─────────────╯
```

## Setup

Install the dependencies:

```bash
npm install
```

## Get started

Start the dev server:

```bash
npm dev
```

Build the app for production:

```bash
npm build
```

Preview the production build locally:

```bash
npm preview
```
